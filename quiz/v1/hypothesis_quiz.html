Quiz System

Total Points: 0

Directions:

1. Custom Feedback Input Field:

When the user selects "Incorrect," a text field appears (custom-feedback-{index}) to allow custom feedback input. This is controlled by the showFeedbackField function.

2. Custom Feedback Handling:

In the calculatePoints function, the user's custom feedback is prioritized over the pre-loaded feedback. If the custom feedback field is empty, it defaults to the pre-loaded feedback for incorrect answers.

3. Feedback Download:

When downloading feedback as .txt, the system checks if custom feedback was provided and includes that in the downloaded file.

This setup allows users to provide their own feedback for incorrect answers, which is then incorporated into the generated feedback file.
