Visualization 1 Description

This scatter plot shows the relationship between car acceleration and country of origin, where each point represents an individual car. I used a scatterplot seperated by country for readability since each country of origin had a large number of cars. Using the raw data provided a full picture to make it as easy to compare the countries as possible.

Cereal Manufacturers by Average Calorie Count

Visualization 2 Description

This horizontal bar chart shows the average calories of different cereals organized by the manufacturer. The visualization shows which manufacturers offer higher or lower calorie count cereals in a simple easy to read format without needing to parse through a complex set of data or overly busy visualization, but still offers more information when hovering your cursor over any of the bars.

Film Genres by Number of Awards

Visualization 3 Description

This visualization is a pie chart showing the number of awards won by different genres of films. The pie chart seperates each genre by color, and the size represents the number of awards won. Hovering your curser over any part of the pie chart will show more information about the films in that genre.

Grocery Store Chains by Amount Spent by Income Bracket

Visualization 4 Description

This visualization shows the total amount of money spent at different grocery store chains. It is a stacked bar chart, each color within a bar on the chart representing different income brackets of the shoppers. This shows both the most popular or most expensive store chains, as well as the demographic that shops there most regularly.
